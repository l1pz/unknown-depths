# Unkown Depths
A very simple rouge-like made in Moonscript (a dynamic scripting language that compiles into Lua), using LÃ–VE, a Lua 2D game engine.

## Features
- random level generation
- ability to shoot
- particles
- random enemies
- main menu
- music

### Customizable color scheme
Every sprite is black and white, so you can color the anyway you want using base16 color themes. There are plenty of them already included in assets/colors.
